# STUDENT MANAGEMANT SYSTEM###
---
## This is a CLI based application that lets the user to perform a CRUD Create,Read,Update and Delete on the Students data achieved through basic concepts of OOPS and file handling.


## How to use:
Using this application is easy and straight forward. You will be presented with the different options and you need to choose those options in order to perform any action.
---
Options:

      1.Add student Record
      2.Edit the Current Record
      3.Delete the Record
      4.View the Record
      5.Exit the program 

---
## Installaion
1. Make sure you have Python installed.
2. Clone the repository or download the files.
  Code for cloning is **git clone https://github.com/hexa-mal/CSV_Reader.git**
3. Open the terminal in the project folder.
4. Run the program:
  Code = **python sms.py**
