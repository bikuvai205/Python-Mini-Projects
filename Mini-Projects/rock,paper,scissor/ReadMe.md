# Rock-Paper-Scissors Game

A simple Python project where the player can play **Rock, Paper, Scissors** against the computer.  
This is a beginner-friendly project to practice Python basics, loops, conditionals, and randomization.

---

##  How to Play

1. Run the Python script (`rock_paper_scissors.py`).
2. Choose your option:
   - `1` = Rock
   - `2` = Paper
   - `3` = Scissors
3. The computer will randomly choose its move.
4. The game will display who won or if itâ€™s a draw.
5. You can continue playing until you choose to quit.

---
##  Rules

- **Rock beats Scissors**
- **Paper beats Rock**
- **Scissors beats Paper**
- Same choices result in a **draw**

---

2. Download or clone the repository.
